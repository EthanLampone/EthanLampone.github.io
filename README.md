## Step 1: Project Planning:
The robot that we will eventually be creating is going to need a couple things to be able to function and actually "move around" and do what it should do. We needed to:
  - Create a hospital laytout for the robot to traverse
  - Create the different room types that the robot can/will visit
  - Assign each rooms a **Priority Type**. Essentially, when the robot is given a certain number of rooms to visit, it should visit rooms with a higher priority fist over those with less priority
  - Come up with different ways the robot can fail if not prompted the right way or a given input file (which is "submitted" to the robot to run) has the wrong syntax.
